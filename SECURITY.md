# Security Policy

Thank you for taking the time to responsibly disclose a potential security issue in this project.

## Reporting a Vulnerability

If you discover a vulnerability, please report it by creating a GitHub Security Advisory for this repository, or opening a new GitHub issue labeled `security` (do not include sensitive details in public issues). 

When you report, please include:
- Which part of the project is affected.
- The version you found the vulnerability in (if applicable).
- A clear description of the issue and possible impact.
- Steps to reproduce the vulnerability.
- Any other details, such as environment, logs, or suggestions for a fix.

I will respond as quickly as I can, investigate, and coordinate a resolution if necessary. Thank you for helping keep the project safe.
